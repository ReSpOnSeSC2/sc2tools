# SC2Replay-Analyzer -- Python dependencies (pinned).

# installs. The Windows .exe installer (packaging/installer.nsi) ships

# versions would make every machine slightly different.

#   1. Edit the version number on the line below.
#   2. Run packaging/build-installer.ps1 -Test to install into a clean
#      virtualenv on Windows and run the smoke suite.
#   3. Update CHANGELOG.md under [Unreleased] -> Changed.
#
